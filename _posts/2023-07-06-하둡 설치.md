## 하둡 설치

* 구축 환경
AWS EC2 - Free Tier
Ubuntu

1. AWS에서 계정 생성, AWS Management Console - EC2
2. 인스턴스 시작으로 생성. 기본값
3. 해당 인스턴스의 Public IP를 복사해서 터미널에서 ssh 명령어를 통하여 접속
{% highlight html %}
  $ ssh -i ./MY_KEY.pem ec2-user@PUBLIC_IP
{% endhighlight %}
